BKO Request

BKO Request adalah fitur untuk menghandle sistem permintaan Security Backup pada saat kekurangan security dalam suatu acara yang membutuhkan pengamanan. Gambar 1 merupakan tampilan form di backend menggunakan Odoo 13.0 Community, Gambar 2 merupakan script untuk menampilkan gambar 1, dan Gambar 3 merupakan API untuk digunakan oleh Android. Fitur ini sudah diterapkan pada perusahaan Interteknis Surya Terang (IST)

Dashboard Internal IST

Dashboard Internal IST (Interteknis Surya Terang) berfungsi untuk menampilkan informasi terkini mengenai patroli, incident, dan informasi karyawan yang tidak masuk tanpa keterangan secara real-time beserta titik koordinatnya. Untuk menampilkannya pada tampilan maps digunakan Google Maps API yang disediakan oleh google.

Monitoring Parkir

BIOPARKS merupakan singkatan dari Biometric Parking System. Sistem Monitoring Parkir ini didesain untuk keperluan monitoring parkiran secara jarak jauh dengan menerapkan konsep IoT. Informasi yang ditampilkan adalah jumlah kendaraan masuk, keluar, sedang parkir dan sisa slot parkiran, serta detailnya yang ditampilkan pada tabel. Informasi yang ditampilkan juga diupdate secara real-time menggunakan Javascript
